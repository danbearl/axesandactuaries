    2. Customize player name
    3. Player to player messaging
    4. Player to player trades
    5. Message Board
    6. Abuse prevention features
    7. Player Avatars
2. Gamification incentives
    1. Ranking
    2. Achievements
    3. Titles
    4. Badges
    5. Property Customizations
    6. Adventurer skins / uniforms
    7. Rare / custom avatars
3. Game Mechanics
    1. Adventurer Equipment
    2. Contract class requirements
    3. Exploitation prevention
    4. Vacation Mode - allows players to pause for periods of time without penalty, i.e., wage and maintenance collection is paused, but no game actions or progress can be made.
    5. Regions
        1. Multiple regions in which players can operate
        2. Different flavors / difficulty levels for each region
        3. Different adventurer pools for each region
        4. Players must expand to a region by setting up offices there before they can accept or execute contracts in that region
        5. World Map
    6. Legendary contracts affect game world in some way - introduce story elements.
    7. Limit number of adventurers by dorm space
    8. Quality of properties influences adventurer loyalty
    9. Increase properties such that there is a bonus building for each class and its primary stat
    10. Determine additional effects for each of the following:
        1. Loyalty
            1. affects whether the adventurer will stay through hard times
            2. Low loyalty may cause an adventurer to sponatneously leave
        2. Ambition
            1. Affects xp gain -- more ambitious adventurers will advance more quickly
            2. Risks quitting if a player's reputation drops
        3. Disposition
            1. Affects party cohesion during contract execution
    11. Limit party size for contracts
    12. Contract completion reports
4. Aesthetics / UX
    1. Visual representation of players' bases/properties
    2. Adventurer portraits
        1. Head Shots
        2. Full Body
        3. Customizeable
        4. Random with very low chance for duplicates
    3. Adventurer profile pages - View adventurer stats, level progression, injuries, etc.
    4. A wiki or some similar documentation on races, classes, characteristics, etc
5. Monetization
    1. Ads?
    2. Subscription - what's the benefit?
    3. Cosmetic purchases
    4. Event Tickets / Season Passes
6. Administrative
    1. Create user roles
        1. Administrator - Able to modify game state, delete player accounts, etc.
        2. Moderator - Provided with tools to moderate social aspects of the game, cannot modify game state or permanently affect players
        3. Players - Can play the game without any elevated priviledges or capabilities
7. Security and Compliance
    1. Ongoing secure code review
    2. Implement secure code best practices
    3. U.S. and E.U. privacy regulatory compliance
    4. High security regarding player data
